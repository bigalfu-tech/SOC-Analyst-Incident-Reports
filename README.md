# SOC Analyst Incident Response Portfolio

This repository contains **25 documented security incident investigations**, demonstrating hands-on SOC skills in:

- Phishing analysis  
- Malware detection and remediation  
- Brute force and password spray detection  
- Reconnaissance and network monitoring  
- Privilege abuse and insider threat investigations  

All cases follow a **consistent reporting format**:

1. **Title** – The incident alert name and type  
2. **Time** – When the alert was triggered  
3. **Summary** – High-level overview of the event  
4. **Analysis** – Technical investigation and reasoning  
5. **Classification** – Incident type, severity, confidence, disposition (True/False Positive)  
6. **Recommendation Action** – Steps for remediation and prevention
