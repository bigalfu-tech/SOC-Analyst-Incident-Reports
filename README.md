# SOC-Analyst-Incident-Response-Reports
This repository contains documented security incident investigations based on simulated SOC scenarios.

## Objective
To demonstrate hands-on skills in:
- Phishing analysis
- Threat detection and investigation
- Incident classification and response

## Tools Used
- VirusTotal
- OSINT techniques
- Log analysis (simulated)

## Incident Categories
- Reconnaissance = gathering information
- Brute Force = guessing passwords
- Phishing = tricking users
- Malware = infected systems doing bad things
- Suspicious Activity = gaining or abusing power

## Methodology
Each case follows a structured approach:
1. Summary of the alert
2. Technical analysis
3. Classification (severity, confidence, disposition)
4. Recommended response actions
